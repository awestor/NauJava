Инструкция к запуску:
Для запуска проекта в докере требуется создать «.env» файл, что будет содержать такие переменные как:
# Для БД
DB_NAME, DB_USER, DB_PASSWORD, DB_PORT
# Для Redis
REDIS_PORT, REDIS_PASSWORD (Можно пустым)
# Для самого приложения
APP_PORT, SPRING_PROFILE, APP_ENV, APP_NAME
# Для работы JWT
JWT_SECRET
# Для логов
LOG_LEVEL
А после в консоли прописать docker-compose up [-d] для запуска сборки контейнера.

В случае разворачивания на локальной машине необходимо поднять редис либо в докере, либо на локальной машине, а также указать:

DB_URL;
DB_USER (по умолчанию = postgres);
DB_PASSWORD (по умолчанию = password);
JWT_SECRET;
REDIS_HOST (по умолчанию = localhost);
REDIS_PORT (по умолчанию = 6379);

В случае изменения БД необходимо указать также и новый драйвер

При первом запуске приложения автоматически создаётся пользователь со всеми правами и данными: "firstUser", "firstUser@example.com", "Password123!"
